# AIML-MRMS Corrected Reproducibility Package v1.1

This repository supports the IJSE-reframed AIML-MRMS manuscript:

**A Governance-Aware Sustainable Engineering Decision Methodology for Mineral Resource Management: An Adaptive AI-MCDM-Optimisation Architecture**

The package documents and reproduces the defensible proof-of-concept outputs from the AIML-MRMS computational workflow. The contribution is framed as a governance-aware sustainable engineering decision methodology and architecture, not as a new SVM, AHP, TOPSIS, or optimisation algorithm.

## Integrity Notice

The former numerical Table 8/9 cross-configuration PCI/RPCI gain comparison has been withdrawn from the manuscript pathway. An integrity audit found that the earlier comparative delta-PCI values were not generated by the computational pipeline and should not be used as performance-superiority evidence.

This corrected release retains the computed SVM, feature-weight, TOPSIS, baseline PCI/RPCI diagnostic, and investment-allocation outputs. It does not support claims that AIML-MRMS empirically outperforms standalone AI, standalone MCDM, or static AI-MCDM configurations across PCI/RPCI gain benchmarks.

## Scope

This package supports:

- Table 5: SVM leave-one-out validation metrics.
- Table 6: SVM feature weights and aggregated criterion signals.
- Table 7: AHP-TOPSIS project rankings and corrected Cycle 3 closeness coefficients.
- Baseline PCI/RPCI diagnostics computed directly from Eqs. 5-6.
- Investment-allocation demonstration using the corrected TOPSIS outputs.
- Qualitative architecture comparison of AIML-MRMS against less integrated decision workflows.

This package does not support:

- Claims that AIML-MRMS has proven empirical performance superiority over baseline configurations.
- Earlier numerical claims such as `NGA delta-PCI = +0.030` and `ZAF delta-PCI = +0.017` as comparative performance evidence.
- Broad empirical generalisation beyond the 16-country African mining proof-of-concept dataset.

## Data Sources

The proof-of-concept uses public, aggregate country-level indicators:

- World Bank Worldwide Governance Indicators, 2022.
- Fraser Institute Annual Survey of Mining Companies, 2022.
- UNCTAD World Investment Report FDI annex values used for PCI/RPCI diagnostics.

No personal or human-subject data are included.

## Running the Pipeline

Install dependencies:

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
```

Run the full pipeline:

```bash
python scripts/05_run_full_pipeline.py --permutations 1000
```

For a faster smoke test:

```bash
python scripts/05_run_full_pipeline.py --permutations 20
```

Outputs are written to `results/tables/`.

## Validation Boundary

The current evidence demonstrates computational traceability and coherent proof-of-concept scenario behaviour using public African mining-governance data. It should not be read as external validation, broad predictive generalisation, or empirical proof that AIML-MRMS outperforms all baseline alternatives.
